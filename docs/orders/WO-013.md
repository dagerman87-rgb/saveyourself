# WO-013: 마을의 숨과 소리

## 배경

운영자 피드백: 그래픽 방향은 합격, 다만 상시 켜두는 화면이므로 마을의 밀도를 올린다 — 울타리, 초원, 나무, 목장 같은 환경 디테일. 여기에 CC0 음악과 앰비언트 사운드를 더해 "틀어두고 싶은 화면"으로 만든다. WO-012 잔여(자연 이벤트 weather 예시, routines.json 전문 재제출)도 이 보고에 포함할 것.

## 작업 범위

### 1. 환경 디테일 (연출 계층 — 세계 데이터 아님)

- 지면: 단색 → 초원 질감(풀 톤 변주, 풀포기·들꽃 점묘), 길은 밟힌 흙 질감 유지. 구역별 결 차이 — 언덕은 짙은 풀, 항구는 모래 섞임, 갯벌은 뻘·갈대
- 울타리와 경계: 사과나무 마당 울타리 보강, 텃밭·목장 울타리, 밭두렁. 구역 경계는 벽이 아니라 지형의 결로 (언덕 단차, 갯벌 물웅덩이)
- 식생: 언덕에 소나무 3~5, 마을 곳곳 잡목·덤불, 갯벌 갈대숲, 바위 몇 점
- 생활 소품: 빵집 뒤 텃밭(이랑+채소), 부두의 통·그물 걸이·상자, 시장 뒤 수레, 집들 사이 빨랫줄(빨래는 낮에만), 우물 하나(항구 골목께)
- 작은 목장: 구시가 비탈에 울타리 친 마당 — 염소 2~3 + 빵집 뒤 닭 서너 마리. 느리게 배회하는 앰비언트 동물(연출 전용 — 세계 데이터에 없음. 엔진이 서사에서 가축을 언급하면 그쪽이 정본, 뷰어가 따라감)
- 하늘의 생명: 갈매기 2~3이 부두 위를 이따금 선회 (예론이 이름 붙인 그 갈매기들)
- 취침 연출 수정 (지난 확인 반영): 취침 시 캐릭터가 건물 안으로 사라지고 지붕 위 💤 만 표시. 기상 시 문에서 나오는 연출

### 2. 소리

- 음악 (CC0 한정): 잔잔한 마을 앰비언트 BGM — 낮/밤 각 1곡 이상, 루프, 교차 페이드. 출처는 CC0 명시 소스만: FreePD.com, OpenGameArt(CC0 필터), Pixabay 오디오 등. 라이선스 증빙 필수: `viewer/assets/audio/CREDITS.md`에 곡명·출처 URL·라이선스 기록 (CC-BY는 불가 — 표기 의무가 생기므로 CC0/퍼블릭도메인만)
- 앰비언트 사운드: 파도(상시, 은은), 갈매기(낮, 간헐), 귀뚜라미(밤), 비 올 때 빗소리, 대장간 근처 망치 소리(로한이 work 중일 때만). 파일이 마땅치 않으면 WebAudio 합성 허용
- 컨트롤: 상단 바에 음소거 토글 + 볼륨. 기본 볼륨 낮게(배경용). 브라우저 자동재생 정책 대응(첫 클릭 시 시작 안내)

### 3. 성능·절제 원칙

- 상시 화면 전제: 애니메이션 요소 총량 관리, 탭 비활성 시 오디오 페이드아웃+정지
- 디테일은 서사를 가리지 않는다: 소품·동물은 이벤트 재생(이동·말풍선)보다 시각적으로 낮은 존재감. 화면의 주인공은 여전히 주민 6명이다
