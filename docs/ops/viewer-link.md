# 어항 접속 링크

집 밖(회사 등)에서 어항을 보는 주소다. Cloudflare Quick Tunnel은 터널을 재시작할 때마다
주소가 바뀌므로, 이 파일이 링크의 정본이다. 갱신은 `engine\publish-tunnel.cmd`가 한다.

    https://changed-station-remember-parameter.trycloudflare.com/?t=W-PsDvT9M6zMWbsVPl7KUvRy

갱신 시각: 2026-07-24 19:11 (KST)

## 쓰는 법

- 회사 PC에는 아무것도 설치할 필요가 없다. 브라우저로 위 주소만 열면 된다.
- 한 번 열면 토큰이 쿠키로 저장되어, 다음부터는 `?t=...` 없이 주소만으로 열린다.
- 집컴이 켜져 있고 뷰어·터널이 떠 있어야 보인다. 바탕화면 `미르한 어항` 바로가기가 셋 다 챙긴다.

## 주의

- 이 링크를 아는 사람은 **세계의 모든 진실**(비밀·숨은 욕망·텐션)을 보고 **신탁까지 내릴 수 있다.**
  공개 인터넷 주소이므로 토큰이 유일한 방벽이다. 공유에 주의할 것.
- 토큰을 갈려면 `engine\.viewer-token`을 지우고 뷰어를 재시작한 뒤 `publish-tunnel.cmd`를 다시 실행한다.
- 이 저장소는 private이다. 공개 전환 금지 원칙에 따라 이 파일도 밖으로 나가지 않는다.
